Portable LIDAR Unit Planning Utility

This application is useful for planning placements for LIDAR scans of an environment by allowing the user to upload a floorplan and custom place walls/obstacles that would be in the environment for the LIDAR to scan. With additional functionality that the user can place targets for tying multiple LIDAR scan positions together and see in real time which scan positions can see which targets.

Team Task Manager

With this application a user or a team can more efficiently view, assign, and complete tasks. Simply log in and select to either set up a personal scrum board or a team scrum board.

Personal

Keep track of home improvement projects you want to complete over the summer. Create a task list to clean out your garage. Keep yourself accountable and on track.

Team

Invite your coworkers to your scum team. Start adding tasks and either assign them or allow them to be taken. Only you can remove a task from a user and put it back in the backlog or move the task to the finish line but your team can move the item based on whether it is in the to-do, in progress, or blocked phase.
